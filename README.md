# Employee Management System


This system includes login functionality for employees.

This system includes a dashboard to manage employees.


It allows adding, viewing, and managing employee details.

## Branching Strategy (Task 17)

We use the Git Flow branching strategy for managing the Employee Management System project.

- The **main branch** contains stable and production-ready code.
- A **develop branch** is used to integrate all features before release.
- Each developer works on separate **feature branches** such as feature/login, feature/dashboard, etc.
- Bug fixes are handled using **bugfix branches**.
- For production preparation, **release branches** are created.
- Urgent issues in production are fixed using **hotfix branches**.

This strategy helps in better collaboration, parallel development, and ensures stable releases.
