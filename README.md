# Employee Management System

This system includes login functionality and a dashboard to manage employees.

It allows adding, viewing, and managing employee details.

## Branching Strategy (Task 17)
We use the Git Flow branching strategy for managing the Employee Management System project.
- The main branch contains stable and production-ready code.
- A develop branch is used to integrate all features before release.
- Each developer works on separate feature branches.
- Bug fixes use bugfix branches.
- Releases are handled using release branches.
- Urgent fixes use hotfix branches.
